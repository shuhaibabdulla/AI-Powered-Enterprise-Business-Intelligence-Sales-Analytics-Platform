## 📌 Project Overview

An enterprise-grade, end-to-end Business Intelligence and Analytics platform built on real Brazilian e-commerce data (100,000+ orders).

This project demonstrates the complete data analytics pipeline:

**Raw Data → Data Cleaning → Exploratory Data Analysis (EDA) → KPI Development → Machine Learning Forecasting → Interactive Dashboards**

---

## 🎯 Business Problem

RetailX Corp needed a unified analytics platform to:

- Monitor revenue, orders, and customer trends in real time
- Identify underperforming product categories
- Reduce late delivery rates
- Forecast next month's revenue
- Generate AI-powered executive recommendations
